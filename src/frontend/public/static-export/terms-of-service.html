t09n.com is a professional browser-based translation platform that provides:

  • A computer-assisted translation (CAT) editor with segment-level editing capabilities
  • A proprietary dual-layer translation system that provides automated translation suggestions
  • Translation memory (TM) storage, management, and leverage
  • Terminology base management and glossary enforcement
  • Translation Quality Assessment (TQA) reporting tools
  • Multi-format document processing (DOCX, XLSX, PPTX, PDF, HTML, TXT, and more)
  • Native support for 10+ Indian languages including Hindi, Tamil, Telugu, Kannada, Malayalam, Bengali, Marathi, Gujarati, Punjabi, and Odia
  • Project management and collaboration tools
