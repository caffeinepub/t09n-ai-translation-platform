ASSETS FOLDER — t09n.com Static Export
=======================================

Copy the following files from your source project into this folder:

FROM: src/frontend/public/assets/uploads/
  TO: assets/uploads/

Required files:
  Version-01-1-1.png  — favicon
  Version-01-2-1.png  — navbar logo
  Version-01-3-1.png  — hero logo (inverted)
  Version-04-1.png    — footer & legal pages logo
  Version-06-1.png    — brand wordmark (optional)

FROM: src/frontend/public/assets/
  TO: assets/

Required files:
  Fontspring-DEMO-allroundgothic-boldoblique.otf  — hero font

Without these files the site will still work, but logos will be broken
and the hero font will fall back to system fonts.
